# Presentación (Web) · Marketplace Multirrol

Incluye:
- Logo (SVG)
- Paleta de colores definida
- 2 tipografías (Inter + Poppins)
- 4 íconos personalizados (SVG)
- 4 productos demo
- Tema de la tienda (estilo minimalista moderno)
- 2 páginas web de referencia

## Guardado en cache (localStorage)
Al abrir `index.html`, el contenido se guarda en el cache del navegador en formato JSON usando esta clave:

`cuentica_marketplace_presentation_v1`

Puedes:
- **Exportar JSON** (descarga el archivo con lo que está en cache)
- **Ver JSON** (descarga un preview)
- **Limpiar cache** (borra la clave)

## Ejecutar
Solo abre `index.html` en tu navegador.
